# 💳 Credit Scoring Model
### CodeAlpha Machine Learning Internship — Task 1

## 📌 Objective
Predict an individual's **creditworthiness** using past financial data.

## 🛠️ Technologies Used
- Python 3.x
- Scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn

## 🤖 Models Used
| Model | Description |
|---|---|
| Logistic Regression | Baseline linear classifier |
| Decision Tree | Rule-based tree model |
| Random Forest | Ensemble of decision trees |

## 📊 Evaluation Metrics
- Accuracy
- Precision, Recall, F1-Score
- ROC-AUC Score
- Confusion Matrix

## 📁 Dataset
Synthetic financial dataset with features:
- Age, Income, Loan Amount, Loan Duration
- Number of Credits, Employment Years
- Debt Ratio, Payment History, Credit Score
